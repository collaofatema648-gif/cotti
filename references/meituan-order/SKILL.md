在美团App中找到店铺 → 选择饮品和规格 → 支付 → 等待配送

## 技术实现

本 Skill 通过以下方式工作：

1. **信息提供**：调用 MCP 工具获取实时店铺信息、优惠活动、饮品推荐
2. **引导用户**：提供美团 App 使用指引
3. **深度链接**：生成美团店铺链接或二维码（需配置店铺ID）
4. **不涉及**：不处理订单、支付、配送等实际业务（这些在美团 App 中完成）

### 配置说明

编辑 `config.json` 文件：

```json
{
  "store_name": "库迪咖啡（IC设计产业园店）",
  "store_id": "替换为实际的美团店铺ID",
  "meituan_deep_link": "meituan://shop?shopId=YOUR_SHOP_ID",
  "meituan_web_link": "https://www.meituan.com/shop/YOUR_SHOP_ID",
  "meituan_search_keyword": "库迪咖啡 IC设计产业园店"
}
```

**如何获取店铺ID：**
1. 打开美团 App
2. 搜索"库迪咖啡 IC设计产业园店"
3. 进入店铺页面
4. 分享店铺链接，从链接中提取店铺ID

## 使用示例

```python
# 获取店铺信息
python meituan_order.py --action store_info

# 获取美团链接
python meituan_order.py --action links

# 生成店铺二维码
python meituan_order.py --action qrcode --link "https://www.meituan.com/shop/XXX"

# 查看点单指引
python meituan_order.py --action guide

# 获取饮品推荐
python meituan_order.py --action recommend
```
